## 🛠️ If Something Goes Wrong

If the app does not start:

- Download the file again
- Make sure the download finished
- Try running it as a normal user first
- Check that Windows is not blocking the file

If the SD card is not found:

- Remove the card and insert it again
- Try a different USB port
- Try a different reader
- Close other apps that may be using the card

If the write process stops:

- Check that the card has enough space
- Make sure the card is not locked
- Close any file explorer windows that open the card
- Start the process again

If SS1 does not read the card:

- Remove the card and insert it again
- Confirm the image finished writing
- Try another SD card if needed
- Check the card in Windows to make sure it appears normally
